# Docs báo cáo

Thư mục này dùng để lưu toàn bộ nội dung phục vụ viết báo cáo khoa học trên Codex.

## Gợi ý tổ chức
- `MUC_LUC_BAO_CAO_CAP_NHAT.md`: mục lục tổng thể (master TOC).
- `chuong-1.md` ... `chuong-5.md`: nội dung chi tiết từng chương (sẽ bổ sung dần).
- `phu-luc.md`: phụ lục tài liệu tham chiếu.

Khi làm việc tiếp theo, chúng ta sẽ đi sâu từng chương và lưu trực tiếp vào thư mục `docs/`.
